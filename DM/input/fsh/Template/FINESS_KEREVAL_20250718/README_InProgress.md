# Note à l'intention de l'équipe NOS uniquement
## Documentation sur le dossier "InProgress" 
Ce Dossier sert à stocker les FSH des ASS, JDV et TRE modifiés dans le mois en cours mais pas encore validés.
**Attention !** Il faudra bien s'assurer à chaque merge de la branche du mois en cours sur le main que ce dossier ne contient plus aucun FSH. Seul le fichier **README_InProgress.md** doit être laissé afin de conserver l'arborescence actuelle et éviter que ce dossier ne soit supprimé (un dossier vide est automatiquement supprimé dans un repository GIT).

Voir le fichier **Readme_Traitement_NOS_FSH.md** pour les étapes à suivre lors du traitement des NOS en FSH dans un mois classique.
